    Commands

      mkdir NAME - creating a folder
      ls -a - list of folder -a means to include even the secret files.
      mkdir .NAME - create a secret folder.
      cd - Changing File Directory. Tab for auto complete Directory 
      cd ~ - going to root folder.
      cd .. - goiing one step back.
      ctrl e - going to the last line of command.
      ctrl u - clear all you are typing.
      touch NAME.txt - touch means to create a file
      start NAME.txt - opens the file in notepad.
      code NAME.txt - opens the file in vscode
      rm NAME.txt - removes the file. 
      rm * - removes everything in the directory.
      rm -r FOLDER - removes the folder or the directory. 
      pwd - Print Working Directory, tells your entire current path.
