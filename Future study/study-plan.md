# Future Study Plan

## 1. Tailwind CSS

- Utility-first CSS framework
- Learn before React so you can style React projects with it right away

## 2. React 18 + TypeScript + Next.js

- React 18 fundamentals, hooks, state management, routing
- React Class Components (still used in older codebases)
- API handling with Axios
- TypeScript basics + React with TypeScript
- Next.js with TypeScript + full-stack projects
- Build a simple static React project first (no database) to check what you've learned
- Then build a full MERN project (MongoDB, Express, React, Node.js)
- 30+ hours of content
- Course: https://www.udemy.com/course/react-tutorial-and-projects-course/?couponCode=KEEPLEARNING

## 3. Ruby + Ruby on Rails

- Ruby language fundamentals
- Rails framework for backend web development

## 4. PHP + Laravel

- PHP language fundamentals
- Laravel framework for backend web development

## 5. JavaScript & Data Structures/Algorithms

- Important for interviews and getting hired
- Should be top priority alongside learning frameworks

## 6. Open Source Contribution

- Contribute to open source projects
- Programs like GSoC, LFX Mentorship
- Builds confidence and looks good on resume
